Rating API Reference¶

hosting.rating API is used to get credits amount rating.

Methods¶

hosting.rating.list(apikey[, opts=nil])¶

List credits consumption by resource

Parameters:
  • apikey (string) – API connection key
  • opts (struct) – Filtering options
Returns:

list of struct – list of consumption
