Hosting Catalog API Reference¶

API is used to check the credit amount cost for hosting component.

Methods¶

hosting.catalog.list(apikey[, opts=nil])¶

List credits consumption by resource

Parameters:
  • apikey (string) – API connection key
  • opts (struct) – Filtering options
Returns:

list of struct – list of consumption
hosting.catalog.price(apikey, items)¶

Retrieve the prices for a given product_type/quantity/flavor/state.

Prices are returned in the same order they where entered.

Parameters:
  • apikey (string) – API connection key
  • items (array) – API connection key
Returns:

list of struct – list of consumption
